Custom Product Libraries

Below is a list of custom product libraries. Each library can be installed directly through the Arduino IDE by searching its name, or manually downloaded from the linked GitHub repository.

For ESP32 microWatt: PTSolns_microWatt

For I2C Backpack: PTSolns_I2CBackpack

For Interface-Shield: PTSolns_InterfaceShield

For Pico Base family: PTSolns_PicoBase

For any BMx280 (BMP280, BME280) sensor module: BMx280

For any AHTx (AHT20, etc) sensor module: PTSolns_AHTx

For any VEML7700 sensor module: PTSolns_VEML7700
